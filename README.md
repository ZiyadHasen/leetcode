## Solution Structure

Each solution folder contains:

- **Solution File**: The main code implementation.
- **Description/Intuition File**: A detailed breakdown of the solution approach, including:
  - Problem statement summary
  - Thought process
  - Different approaches considered
  - Time and space complexity analysis
  - Edge cases handled

## How to Use

1. Clone this repository to your local machine:

   ```bash
   git clone https://github.com/yourusername/leetcode-solutions.git
   cd leetcode-solutions
   ```

2. Navigate to the relevant topic folder to find the problem you are interested in.

3. Review the solution and the accompanying description file to understand the approach.

4. Feel free to run the code and test it with different inputs!
